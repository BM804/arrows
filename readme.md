# 🎯 Arrows

A fun and challenging browser puzzle game built using **HTML, CSS, and JavaScript**.

Clear every arrow path from the board without causing collisions. Each level becomes more difficult with more paths to solve.

---

## 🎮 Gameplay

- Click on an arrow path to launch it.
- A path can only leave the board if the space ahead of its arrowhead is completely clear.
- If another path blocks the way, you lose one life.
- Clear every path to advance to the next level.
- Lose all lives and the game restarts.

## 🎮 How to Play

1. Start the game.
2. Find an arrow whose path to the edge is clear.
3. Click the arrow.
4. Successfully remove every arrow.
5. Advance through increasingly difficult levels.
